## Project 1 - OTT Platform Automation Framework

- **Project Name:** OTT Platform Automation Framework
- **Company:** Source Digital
- **Start Date:** Apr 2024
- **End Date:** Present
- **Domain:** OTT Streaming

### Overview

Dual-script Python automation system for continuous OTT content playback testing and metadata management across major streaming platforms.

### Key Contributions

- Developed Python + Selenium script to automate content playback across different OTT platforms
- Built Python + Playwright script to extract and sync OTT content metadata directly into PostgreSQL database
- Monitor script execution and daily event status via PostgreSQL and Otter dashboards
- Debug issues like selector changes and script failures, raise GitLab Merge Requests for lead review before deployment to remote Windows servers via SSH

### Tech Stack

Python, Playwright, Selenium, PostgreSQL, GitLab, SSH
